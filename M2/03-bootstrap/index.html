Bootstrap

Bootstrap es una biblioteca digital que concentra información y un framework para el diseño de plataformas de código abierto.

La historia de Bootstrap comienza en Twitter. En 2010, dos empleados de esta reconocida red social, Mark Otto y Jacob Thornton, se propusieron desarrollar una colección de herramientas y estilos que les permitiera mantener una consistencia en el diseño de los productos que estaban construyendo.

Para poder utilizarlo en nuestro proyecto nos dirigimos a la documentación de Bootstrap

Y seguimos los siguientes pasos:

  • Copiamos la etiqueta "link" en nuestro header
  • Copiamos las etiquetas "script" al final del body
  • Seleccionamos los componentes deseados y pegamos el/los nombre/s de clase/s a nuestros elementos html
  • Layout y contenedores
  • Componentes
